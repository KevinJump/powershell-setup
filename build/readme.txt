
# Umbraco Template Script

this script will help you setup your project template. 

./build1.ps [method]

Methods 

 [setup]    Handle the basic setup of your project. 

            For this script you will need to know: 

            - the name of your library
            - the path the Umbraco site you want to use for testing

 [package]  package your library up into a nuGet and Umbraco
            zip package
